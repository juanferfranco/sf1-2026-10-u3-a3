# Implementación del temporizador usando p5.js

Para realizar esta actividad tendrás que clonar el repositorio:

1. Abre la terminal Gitbash. En los computadores de la Universidad la tienes disponible. Para instalarla en tu equipo personal necesitarás instalar [Git](https://git-scm.com/)

2. Clone este repositorio con el comando:

```bash
git clone https://github.com/juanferfranco/sf1-2026-10-u3-a3.git
```

3. Una vez clonado el proyecto. Abre el directorio donde se encuentra el proyecto en Visual Studio Code.

4. Instala la extensión [p5.vscode](https://marketplace.visualstudio.com/items?itemName=samplavigne.p5-vscode)

5. Presiona el botón Go Live. Esto te creará un servidor local que permitirá abrir la aplicación en el browser.
